# __GO__(lang)__DO__(it)
##### A simple todo app written in go, (go)ncurses and objectbox.
![Preview of the app (godo)](screenshots/preview.png "godo preview")

| Key     | Action               |
| ------- | -------------------- |
| `space` | Toggle Check           | 
| `j`     | Move cursor up         |
| `k`     | Move cursor down       |
| `o`     | Add New Todo           |
| `d`     | Delete Selected Todo   |
| `q`     | Quit App               |
